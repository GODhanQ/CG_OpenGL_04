@@ 추가된 기능 @@
- 여러 광원조명 지원
- 조명 궤도 반지름 및 회전 모드 조절 기능 추가

- 로봇 이동 기능 개선
- 로봇 소환시 마우스 휠 버튼으로 카메라 회전 가능
- 로봇과 벽의 충돌 처리

- 미로 크기에 맞춰 미니맵 크기 자동 조절
- 미니맵에서 현재 위치 빨간점으로 표시

@@ '바뀐' 키 조작법 @@

' m ' / ' M ' -> ' m ' 으로만 육면체의 움직임 조절
-> 처음 m 키 누르면 높이 움직임 모드
-> 다시 m 키 누르면 멈춤

' s ' -> ' f '로 객체 생성키 바뀜

화살표 객체 움직임 -> wasd 키 조작으로 바꿈

@@ '추가된' 키 조작법 @@

' l ' and ' L ' : 조명 궤도 반지름 조절
-> L : 반지름 증가
-> l : 반지름 감소

' j ' : 조명 회전 모드 토글
-> j 키 누를 때마다 반시계/시계/멈춤 토글

' w ', ' a ', ' s ', ' d ' : 로봇 이동 조작
-> 로봇이 바라보는 방향기준으로 이동

' GLUT_MIDDLE_BUTTON ' : 마우스 휠 버튼 누름 (Toggle)
-> 로봇 소환시 카메라를 마우스로 돌릴 수 있음
