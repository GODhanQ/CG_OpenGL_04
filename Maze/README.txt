@@ '바뀐' 키 조작법 @@

' m ' / ' M ' -> ' m ' 으로만 육면체의 움직임 조절
-> 처음 m 키 누르면 높이 움직임 모드
-> 다시 m 키 누르면 멈춤

' y ' / ' Y ' -> ' y ' 으로만 카메라 회전 조절
-> 처음 y 키 누르면 +y축 회전 모드
-> 두 번째 y 키 누르면 -y축 회전 모드
-> 세 번째 y 키 누르면 멈춤

' s ' -> ' f '로 객체 생성키 바뀜

화살표 객체 움직임 -> wasd 키 조작으로 바꿈

@@ '추가된' 키 조작법 @@

로봇 조작 모드 X {
'z' / ' Z ' : 카메라 시점 이동 조절
-> z : 원점 기준 +방향으로 시점 이동
-> Z : 원점 기준 -방향으로 시점 이동

' x ' / ' X ' : 카메라 시점 이동 조절
-> x : 원점 기준 +방향으로 시점 이동
-> X : 원점 기준 -방향으로 시점 이동

'   ' : 스페이스바 : 카메라 시점 이동 조절
-> 스페이스바 : 원점 기준 y축 +방향으로 시점 이동
-> Shift + 스페이스바 : 원점 기준 y축 -방향으로 시점 이동
}

' l ' / ' L ' : 조명 궤도 반지름 조절
-> l : 반지름 감소
-> L : 반지름 증가

' j ' : 조명 회전 모드 토글
-> j 키 누를 때마다 반시계/시계/멈춤 토글

' . ' : 조명 색깔 변경
흰색 -> 빨강 -> 초록 -> 흰색 순서로 변경

' k ' / ' K ' : 로 로봇 광택도 조절
-> k : 광택도 계수 증가
-> K : 광택도 계수 감소

' w ', ' a ', ' s ', ' d ' : 로봇 이동 조작
-> 로봇이 바라보는 방향기준으로 이동

' GLUT_MIDDLE_BUTTON ' : 마우스 휠 버튼 누름 (Toggle)
-> 로봇 소환시 카메라를 마우스로 돌릴 수 있음
